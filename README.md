# Content Creator Agent

A Vite + React + TypeScript prototype for a Chinese content production workspace.

## Features

- Platform-specific creation tasks for Xiaohongshu, Douyin, Shipinhao, Gongzhonghao, and Kuaishou.
- Content forms for article, image-text, and video workflows.
- Admin-managed API Provider setup so creators do not enter API keys.
- Capability Registry and Capability Binding model for connecting creative capabilities to providers.
- Mock content generation flow for draft versions, video prompt packages, and image-text plans.

## Scripts

```bash
npm install
npm run dev
npm test
npm run build
```

The local workspace runs at `http://127.0.0.1:5173/`.
